# E-commerce Analytics Dashboard

Used SQL to extract and transform data from Google BigQuery (TheLook e-commerce dataset) and created a dashboard in Tableau Public. Modeled month-over-month revenue, orders, and returns, and created a 6-month revenue forecast. 

<strong>Tech Stack:</strong>
- Cloud warehouse: **Google BigQuery** 
- Processing: **SQL**
- Dashboard: **Tableau Public**
